# 🛰️ RADAR — Como abrir com 2 cliques (PC e celular)

Essa versão **não usa Node, não usa npm, não usa terminal**. É só HTML/CSS/JS puro
(mesmo esquema do seu app do Espetinho do Batata no GitHub Pages). O progresso fica
salvo no navegador (localStorage) do dispositivo que você usar.

---

## 💻 No computador — 2 cliques, sem internet

1. Descompacte a pasta `RADAR_TRAINER_WEB` em qualquer lugar (Desktop, por exemplo).
2. Dê **duplo clique** no arquivo **`Abrir RADAR.bat`**.
3. Pronto — abre direto no seu navegador padrão.

Se quiser ainda mais rápido: clique com o botão direito no `Abrir RADAR.bat` →
**Enviar para → Área de trabalho (criar atalho)**. Aí fica um ícone na sua Desktop,
igual um programa normal.

> Isso funciona 100% offline, não precisa instalar nada.

---

## 📱 No celular — vira um "app" de verdade (recomendado)

Como o celular não acessa arquivos do seu PC diretamente, o jeito mais fácil é
hospedar essa pasta *de graça* no GitHub Pages (a mesma conta `Tedgrei` que você
já usa) e depois instalar como atalho no celular. Uma vez feito, é **1 toque** pra abrir.

### Passo 1 — Subir pro GitHub Pages
1. Crie um repositório novo no GitHub (ex: `radar-trainer`).
2. Envie todo o conteúdo da pasta `RADAR_TRAINER_WEB` pra esse repositório
   (pelo site do GitHub: "Add file → Upload files", arraste tudo — ou via `git push` se preferir).
3. Vá em **Settings → Pages** do repositório.
4. Em "Branch", selecione `main` e pasta `/root`, clique em **Save**.
5. Em alguns minutos seu app estará em:
   `https://tedgrei.github.io/radar-trainer/`

### Passo 2 — Instalar no celular (1 toque depois disso)
1. Abra esse link no navegador do celular (Chrome no Android, Safari no iPhone).
2. **Android (Chrome):** toque nos 3 pontinhos → **"Instalar aplicativo"** (ou
   "Adicionar à tela inicial"). Vai aparecer um ícone do RADAR na sua tela, igual
   um app da Play Store.
3. **iPhone (Safari):** toque no ícone de compartilhar (o quadrado com seta) →
   **"Adicionar à Tela de Início"**.
4. Pronto — toque no ícone sempre que quiser estudar, abre em tela cheia, sem barra
   de navegador, como um app nativo.

### Passo 3 — Instalar no computador também (opcional, mesmo link)
No Chrome/Edge do PC, abra o mesmo link e clique no ícone de **instalar** que
aparece na barra de endereço (ou menu → "Instalar RADAR"). Cria um atalho na área
de trabalho/menu iniciar, abre como janela própria, sem abas do navegador.

---

## ⚠️ Atenção: progresso é por dispositivo

Como o progresso fica salvo no navegador local (não em um servidor), se você
estudar pelo PC e depois abrir pelo celular, o progresso **não é o mesmo** —
cada aparelho guarda o seu. Escolha um dispositivo principal pra acompanhar,
ou refaça o check nos dois se for usar os dois.
